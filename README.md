# 基于复合光线追踪的贝叶斯-拟梯度下降镜场布局优化
- 2023 全国大学生数学建模竞赛，A 题送评 **国一**
- 编程亮点: 利用 numpy 构造 7 维张量, 最大化并行运算的加速效果
- 前置知识: 空间几何, 方程求解 (sympy), 贝叶斯优化 (optuna), 梯度下降法 (numpy)
